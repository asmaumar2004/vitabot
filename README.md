## Setup Instructions

To set up the project locally, follow these steps:

1. **Clone the repository:**
    ```bash
    git clone https://github.com/humaimaa/vitabot.git
    ```

2. **Navigate to the project directory:**
    ```bash
    cd vitabot
    ```

3. **Install the required dependencies:**
    ```bash
    npm install
    ```

4. **Run the project:**
    ```bash
    npm start
    ```
